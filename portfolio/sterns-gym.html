Stern's Gym

Local Gym needed a new website to promote business. The previous website was showing age and did not represent the current goals of the Gym. The goal was to improve the user experience, modernize the aesthetics, increase conversions and sell product online.

  • Web Design
  • Site Redesign
  • IU/UX Design
  • Shopify
    • Visit Site

As a web designer, I was in charge of building the complete interface for the new site, redefining the user experience and defining an e-commerce solution.

The main challenge was the integration with a third-party solutions and their adaptation to the Client's needs. An easy to use, well supported platform, and feature rich out of the box.

Right Choice

The solution came to a fully customized theme design for the website with an integrated Shopify store, the client would be able to easily manage and update without too much technical knowledge.

eCommerce Solution

It's easy to use - Shopify has a very friendly administrative dashboard. All of the menus are very straightforward and user-friendly: Catalog, Sales, Pages, etc.

Store Catalog - is easy to update products, does not require technical or coding expertise.

Anyone can use Shopify - it manages to combine powerful eCommerce tools with an easy-to-use editor. Its interface is clean and simple.

Key Results


  • Reliable solution for eCommerce and front store for local business.
  • Scalable application for processing large quantities of sales.

Impact

Stern's Gym is now a 'big WOW' and increased time on site and new business sales inquiries. The site was also SEO optimized and has seen positive rank results.
